# Zoho First Round Preparation Checklist 🎯

## Overview
This checklist covers all essential topics for Zoho's first round aptitude test. Check off each topic as you complete your preparation.

## 📊 Quantitative Aptitude

### 🔢 Number Systems
- [ ] **Number Types**
  - [ ] Natural numbers, whole numbers, integers
  - [ ] Rational and irrational numbers
  - [ ] Prime and composite numbers
- [ ] **Number Properties**
  - [ ] Divisibility rules (2, 3, 4, 5, 6, 8, 9, 10, 11)
  - [ ] LCM and HCF/GCD
  - [ ] Factorization
- [ ] **Number Operations**
  - [ ] BODMAS/PEMDAS
  - [ ] Fractions and decimals
  - [ ] Percentage conversions

### 🧮 Algebra
- [ ] **Basic Algebraic Operations**
  - [ ] Linear equations (one variable)
  - [ ] Linear equations (two variables)
  - [ ] Quadratic equations
- [ ] **Algebraic Identities**
  - [ ] (a+b)², (a-b)², (a+b)(a-b)
  - [ ] (a+b)³, (a-b)³
  - [ ] a³+b³, a³-b³
- [ ] **Inequalities**
  - [ ] Linear inequalities
  - [ ] Quadratic inequalities

  - [ ] Simple average/mean
  - [ ] Weighted average
  - [ ] Average speed problems
- [ ] **Applications**
  - [ ] Age-related averages
  - [ ] Score-related averages
  - [ ] Combined averages

### 📊 Logarithms
- [ ] **Fundamentals**
  - [ ] Definition and properties

- [ ] **Applications**
  - [ ] Change of base formula
  - [ ] Exponential equations
  - [ ] Growth and decay problems

- [ ] **Geometric Progression (GP)**
  - [ ] nth term formula
  - [ ] Sum of n terms
  - [ ] Infinite GP sum
- [ ] **Harmonic Progression (HP)**
  - [ ] Basic concepts

## 🎯 Quick Tips for Success

1. **Time Management**: Practice with a timer for each topic
2. **Formula Sheet**: Create a quick reference formula sheet
3. **Weak Areas**: Spend extra time on topics you find challenging
4. **Mock Tests**: Take at least 2-3 full-length tests per week
5. **Programming**: Focus on logic and basic syntax for first round
6. **Revision**: Last week should be pure revision and practice tests
