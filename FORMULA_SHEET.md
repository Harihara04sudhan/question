## 🔢 **NUMBER SYSTEMS**

### Basic Operations
- **LCM × HCF = Product of two numbers**
- **Divisibility by 3:** Sum of digits divisible by 3
- **Divisibility by 9:** Sum of digits divisible by 9
- **Divisibility by 11:** Alternating sum of digits divisible by 11

### Number Properties
- **Even numbers:** 2n
- **Odd numbers:** 2n + 1
- **Sum of first n natural numbers:** n(n+1)/2
- **Sum of first n even numbers:** n(n+1)
- **Sum of first n odd numbers:** n²

## 🧮 **ALGEBRA**

### Linear Equations
- **ax + b = 0** → x = -b/a
- **ax + by + c = 0** (Standard form)

### Quadratic Equations
- **ax² + bx + c = 0**
- **Discriminant (D):** b² - 4ac
- **Roots:** x = (-b ± √D)/2a
- **Sum of roots:** -b/a
- **Product of roots:** c/a

### Important Identities
- **(a + b)² = a² + 2ab + b²**
- **(a - b)² = a² - 2ab + b²**
- **(a + b)(a - b) = a² - b²**
- **(a + b)³ = a³ + 3a²b + 3ab² + b³**
- **(a - b)³ = a³ - 3a²b + 3ab² - b³**
- **a³ + b³ = (a + b)(a² - ab + b²)**
- **a³ - b³ = (a - b)(a² + ab + b²)**

## 📈 **AVERAGES**

### Basic Formulas
- **Average = Sum of all values / Number of values**
- **Weighted Average = (w₁x₁ + w₂x₂ + ... + wₙxₙ)/(w₁ + w₂ + ... + wₙ)**
- **Average Speed = Total Distance / Total Time**

### Special Cases
- **When n numbers are increased by x:** New average = Old average + x
- **When n numbers are decreased by x:** New average = Old average - x
- **Combined Average:** (n₁A₁ + n₂A₂)/(n₁ + n₂)

## 📊 **LOGARITHMS**

### Basic Properties
- **log(ab) = log(a) + log(b)**
- **log(a/b) = log(a) - log(b)**
- **log(aⁿ) = n × log(a)**
- **log(ⁿ√a) = (1/n) × log(a)**
- **logₐ(b) = log(b)/log(a)** (Change of base)

### Special Values
- **log(1) = 0**
- **log(10) = 1** (for log base 10)
- **ln(e) = 1** (for natural log)
- **aˡᵒᵍₐ⁽ˣ⁾ = x**

## 🔄 **PROGRESSIONS**

### Arithmetic Progression (AP)
- **nth term:** aₙ = a + (n-1)d
- **Sum of n terms:** Sₙ = n/2[2a + (n-1)d]
- **Sum of n terms:** Sₙ = n/2[first term + last term]
- **Common difference:** d = aₙ₊₁ - aₙ

### Geometric Progression (GP)
- **nth term:** aₙ = a × rⁿ⁻¹
- **Sum of n terms:** Sₙ = a(rⁿ - 1)/(r - 1) [r ≠ 1]
- **Sum of infinite GP:** S∞ = a/(1-r) [|r| < 1]
- **Common ratio:** r = aₙ₊₁/aₙ

### Harmonic Progression (HP)
- **If a, b, c are in HP, then 1/a, 1/b, 1/c are in AP**
- **Harmonic mean of a, b:** 2ab/(a+b)

## ⏱️ **TIME, SPEED, AND DISTANCE**

### Basic Formulas
- **Speed = Distance/Time**
- **Distance = Speed × Time**
- **Time = Distance/Speed**
- **Average Speed = Total Distance/Total Time**

### Relative Speed
- **Same direction:** Relative speed = |v₁ - v₂|
- **Opposite direction:** Relative speed = v₁ + v₂

### Special Cases
- **Train crossing a pole:** Time = Length of train/Speed of train
- **Train crossing a platform:** Time = (Length of train + Length of platform)/Speed

### Boats and Streams
**Given: Boat speed in still water = u km/hr, Stream speed = v km/hr**
- **Speed downstream = (u + v) km/hr**
- **Speed upstream = (u - v) km/hr**

**Given: Downstream speed = a km/hr, Upstream speed = b km/hr**
- **Speed in still water = (1/2)(a + b) km/hr**
- **Rate of stream = (1/2)(a - b) km/hr**

**Additional Relations:**
- **Time downstream = Distance/(u + v)**
- **Time upstream = Distance/(u - v)**
- **If same distance covered:** Time ratio = (u + v):(u - v)**

## 💰 **PROFIT AND LOSS**

### Basic Formulas
- **Profit = Selling Price - Cost Price**
- **Loss = Cost Price - Selling Price**
- **Profit% = (Profit/Cost Price) × 100**
- **Loss% = (Loss/Cost Price) × 100**

### Important Relations
- **SP = CP + Profit = CP(1 + Profit%/100)**
- **SP = CP - Loss = CP(1 - Loss%/100)**
- **CP = SP/(1 + Profit%/100)**
- **CP = SP/(1 - Loss%/100)**

### Discount Formulas
- **Discount = Marked Price - Selling Price**
- **Discount% = (Discount/Marked Price) × 100**
- **SP = MP(1 - Discount%/100)**

## ⚖️ **RATIO AND PROPORTION**

### Basic Formulas
- **If a:b = c:d, then ad = bc**
- **If a:b = c:d, then (a+b):(c+d) = a:c = b:d** (Componendo-Dividendo)

### Proportion Types
- **Direct Proportion:** a₁/a₂ = b₁/b₂
- **Inverse Proportion:** a₁ × b₁ = a₂ × b₂

### Mixture and Alligation
- **Alligation Rule:** (Cheaper quantity)/(Dearer quantity) = (Mean price - Cheaper price)/(Dearer price - Mean price)

## 💸 **SIMPLE AND COMPOUND INTEREST**

### Simple Interest
- **SI = (P × R × T)/100**
- **Amount = P + SI = P(1 + RT/100)**
- **Principal = (100 × SI)/(R × T)**
- **Rate = (100 × SI)/(P × T)**
- **Time = (100 × SI)/(P × R)**

### Compound Interest
- **Amount = P(1 + R/100)ᵗ**
- **CI = P(1 + R/100)ᵗ - P**
- **Half-yearly:** A = P(1 + R/200)²ᵗ
- **Quarterly:** A = P(1 + R/400)⁴ᵗ
- **Difference between CI and SI for 2 years:** P(R/100)²

## 📐 **GEOMETRY & MENSURATION**

### 2D Shapes

#### Triangle
- **Area = (1/2) × base × height**
- **Area = √[s(s-a)(s-b)(s-c)]** (Heron's formula, where s = (a+b+c)/2)
- **Perimeter = a + b + c**

#### Rectangle
- **Area = length × breadth**
- **Perimeter = 2(length + breadth)**
- **Diagonal = √(l² + b²)**

#### Square
- **Area = side²**
- **Perimeter = 4 × side**
- **Diagonal = side√2**

#### Circle
- **Area = πr²**
- **Circumference = 2πr**
- **Area of sector = (θ/360°) × πr²**
- **Length of arc = (θ/360°) × 2πr**

#### Parallelogram
- **Area = base × height**
- **Perimeter = 2(a + b)**

#### Rhombus
- **Area = (1/2) × d₁ × d₂** (d₁, d₂ are diagonals)

#### Trapezium
- **Area = (1/2) × (sum of parallel sides) × height**

### 3D Shapes

#### Cube
- **Volume = a³**
- **Surface Area = 6a²**
- **Diagonal = a√3**

#### Cuboid
- **Volume = l × b × h**
- **Surface Area = 2(lb + bh + lh)**
- **Diagonal = √(l² + b² + h²)**

#### Cylinder
- **Volume = πr²h**
- **Curved Surface Area = 2πrh**
- **Total Surface Area = 2πr(r + h)**

#### Cone
- **Volume = (1/3)πr²h**
- **Curved Surface Area = πrl** (l = slant height)
- **Total Surface Area = πr(r + l)**
- **Slant height = √(r² + h²)**

#### Sphere
- **Volume = (4/3)πr³**
- **Surface Area = 4πr²**

#### Hemisphere
- **Volume = (2/3)πr³**
- **Curved Surface Area = 2πr²**
- **Total Surface Area = 3πr²**

## 🎲 **PROBABILITY**

### Basic Formulas
- **P(E) = Number of favorable outcomes/Total number of outcomes**
- **P(E) + P(E') = 1** (E' is complement of E)
- **0 ≤ P(E) ≤ 1**

### Combined Events
- **P(A ∪ B) = P(A) + P(B) - P(A ∩ B)**
- **P(A ∩ B) = P(A) × P(B)** (for independent events)
- **P(A|B) = P(A ∩ B)/P(B)** (conditional probability)

### Special Cases
- **Dice:** P(any number) = 1/6
- **Coins:** P(Head) = P(Tail) = 1/2
- **Cards:** Total = 52, Hearts/Diamonds/Clubs/Spades = 13 each

## 🔄 **PERMUTATION AND COMBINATION**

### Basic Formulas
- **n! = n × (n-1) × (n-2) × ... × 1**
- **0! = 1**
- **nPr = n!/(n-r)!** (Permutation)
- **nCr = n!/[r!(n-r)!]** (Combination)

### Important Relations
- **nCr = nC(n-r)**
- **nPr = r! × nCr**
- **nC0 = nCn = 1**
- **nC1 = n**

### Special Cases
- **Circular permutation:** (n-1)!
- **Circular permutation (clockwise = anticlockwise):** (n-1)!/2

## 📊 **DATA SUFFICIENCY**

### Approach Strategy
1. **Statement I alone sufficient** → Answer A
2. **Statement II alone sufficient** → Answer B
3. **Both statements together sufficient** → Answer C
4. **Either statement alone sufficient** → Answer D
5. **Both statements together not sufficient** → Answer E

### Key Points
- Don't solve completely, just check sufficiency
- Consider each statement independently first
- Then consider both together if needed

## 🧮 **COORDINATE GEOMETRY**

### Distance and Section Formulas
- **Distance between (x₁,y₁) and (x₂,y₂):** √[(x₂-x₁)² + (y₂-y₁)²]
- **Section formula (internal):** ((mx₂+nx₁)/(m+n), (my₂+ny₁)/(m+n))
- **Midpoint formula:** ((x₁+x₂)/2, (y₁+y₂)/2)
- **Area of triangle:** (1/2)|x₁(y₂-y₃) + x₂(y₃-y₁) + x₃(y₁-y₂)|

### Line Equations
- **Slope = (y₂-y₁)/(x₂-x₁)**
- **Point-slope form:** y - y₁ = m(x - x₁)
- **Two-point form:** (y-y₁)/(y₂-y₁) = (x-x₁)/(x₂-x₁)

## ⚡ **QUICK CALCULATION TRICKS**

### Squares
- **(a+b)² = a² + b² + 2ab**
- **Numbers ending in 5:** (ab5)² = ab(ab+1)25**
- **Example: 35² = 3×4 = 12, so 35² = 1225**

### Multiplication
- **11 × two-digit number:** Insert sum of digits in middle
- **Example: 11 × 23 = 2(2+3)3 = 253**

### Percentages
- **10% = 1/10, 20% = 1/5, 25% = 1/4, 50% = 1/2**
- **33⅓% = 1/3, 66⅔% = 2/3**
